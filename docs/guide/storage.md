# 存储模式

RulEuler 支持两种规则文件存储方式：

| 模式 | 资源前缀 | 说明 |
|------|----------|------|
| db | `dbr:` | 存储在 `ruleuler_rule_file` 表，支持子文件夹 |
| jcr | 无前缀 | 存储在 JCR 仓库（Jackrabbit） |

## 选择建议

推荐使用 db 模式：

- 数据统一存储在 MySQL，便于备份和迁移
- 支持子文件夹组织规则文件
- 与项目其他数据（用户、权限）在同一数据库

## 注意事项

!!! warning
    创建项目时选择存储类型，创建后不可更改。同一项目内不可混用两种存储模式。
